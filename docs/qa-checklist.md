# QA Checklist

人工审核操作指南：`docs/manual-review-guide.md`

- [ ] `python3 scripts/validate_data.py` passes.
- [ ] `python3 scripts/quality_gate.py` passes.
- [ ] Official seed tools are not ranked as ecosystem projects.
- [ ] Curated dataset has at least 50 records.
- [ ] Rejected dataset has at least 20 records.
- [ ] Fallback web records are labeled fallback.
- [ ] Exa status report exists.
- [ ] Static site data exists.
- [ ] Export package builds.
