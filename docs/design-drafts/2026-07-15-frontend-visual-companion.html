4. Backend touch spectrum (for discussion)

A · 零后端

  • 只 site 呈现：CSS/HTML/少量 charts
  • 报告仍 detail 或纯前端 modal
  • 风险最低，最快上线

B · 前端 JS 可动

  • app/render 支持报告 modal + Esc 栈
  • i18n 图表文案
  • 仍不改 data 管道

C · 允许数据/构建

  • 新字段 / 新 API / build 改 schema
  • 非视觉必需则不推荐

5. What this companion is NOT

  • 不是实现计划，也不是批准后的 spec
  • 不推翻 Style B 气质（除非你主动否决）
  • 下一步：逐题 grilling → 2–3 方案 → 分节设计 → 写 spec
