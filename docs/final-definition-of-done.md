# Final Definition of Done

The project is final only when:

1. Repository package is complete and reusable.
2. Normalized, curated, and rejected datasets exist.
3. Source quality is explicit.
4. Official tools and ecosystem projects are separated.
5. Final reports are generated.
6. Static site is usable.
7. Automation templates and Hermes cron prompts exist.
8. Final quality gate passes.
9. `dist/agent-ecoradar-final.zip` is generated.
