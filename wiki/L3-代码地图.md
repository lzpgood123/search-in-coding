## 关键入口

- **全流程更新入口：** `python3 scripts/update_tracker.py [--github-limit N] [--exa-limit N] [--deploy]`
- **仅数据重建：** `python3 scripts/update_tracker.py --skip-collect`
- **仅站点构建：** `python3 scripts/build_site.py`
- **仅部署：** `python3 scripts/deploy_site.py --dest <webroot>`
- **仅测试：** `pytest tests/`

## 修改指引

| 要做什么 | 改哪个文件 | 改哪里 |
|---------|-----------|-------|
| 增加新的搜索源 | scripts/collect_*.py + normalize.py | 添加采集函数 + from_*() 函数 + main() 注册 |
| 调整分类规则 | scripts/normalize.py | CATEGORY_RULES 字典 + categories_for() 函数 |
| 调整评分权重 | config/scoring.yaml | source_weights, category_weights, penalties |
| 调整 curated/rejected 阈值 | config/scoring.yaml 或 scripts/finalize_data.py | ranking 配置块或 DEFAULT_RANKING |
| 修改 UI | site/js/render.js 或 site/js/app.js | renderAll() 函数中的对应渲染函数 |
| 修改样式 | site/styles.css | CSS 自定义属性（:root）和选择器 |
| 添加新报告 | scripts/generate_reports.py | main() 函数中追加新的 markdown 生成 |
| 修改数据字段 | common.py + quality_gate.py | normalize_project_fields() + 检查规则 |
| 调整部署目标 | scripts/deploy_site.py | copytree() 或 dest 路径 |
| 修改前端数据 | scripts/build_site.py | write_json() 调用和 enrich 函数 |
| 批量翻译 summary | scripts/translate_summaries.py | needs_translation()/translate_one()/run_translation() + data/translations-cache/ |
| 修改测试 | tests/test_*.py | 对应的测试用例 |
