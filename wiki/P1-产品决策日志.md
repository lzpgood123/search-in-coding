> 产品从第一天的所有想法、决策、偏好，按时间线记录。

## 产品愿景（固定置顶）

> 一个全自动、零人工干预的 AI Coding Agent 生态追踪索引库。
> 核心定位：既不是 awesome list，也不是研究论文——它是一个持续运行的、可复用的数据管道，每天自动发现、归一化、评分、发布 AI Coding 工具生态数据。

## 用户偏好（固定置顶）

| 维度 | 偏好 | 说明 |
|------|------|------|
| 交互 | 全自动运行 + 每周 LLM 深度分析 | 每日规则采集，每周 LLM 分析评分 |
| 视觉 | 深色主题，简洁信息型设计 | 纯信息展示，无干扰元素 |
| 技术 | 简单优先，零框架前端 | 避免前端工程化复杂度，安全且易部署 |
| 产品哲学 | GitHub 是总仓库，站点是展示面 | 双重保险：完整历史 + 最新结果 |
| 采集源 | 仅 GitHub | 其他采集器保留停用，将来可恢复 |
| 评分 | 100 分制双层评分 | 60 分可量化(每日) + 40 分 LLM 质量(每周) |
| 分析节奏 | 每日采集 + 每周一 03:00 LLM 分析 | 独立 cron，SenseNova + DeepSeek-V4-Flash |
